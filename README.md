### **Backend Setup**
```bash
git clone https://github.com/khangpt2k6/NaviCV.git
cd backend

python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

pip install -r requirements.txt

uvicorn main:app --reload
```

### **Frontend Setup**
```bash
cd frontend

npm install

npm run dev
```
